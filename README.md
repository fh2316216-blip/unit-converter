# 🔢 Unit Converter

A simple desktop application built with **Python** that converts **Feet to Meters** using a clean graphical user interface.

---

## ✨ Features

* Simple and modern GUI
* Fast and accurate conversion
* Lightweight desktop application
* Easy to use
